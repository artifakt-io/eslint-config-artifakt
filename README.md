# ESLint Rules NodeJS

ESLint config file.

## Install

1. Install the config : 
```bash
$ npm install git+ssh://git@gitlab.com:artifakt-tools/eslint-config-artifakt
```

2. Create a `.eslintrc.json` file and copy/paste this `json` code block : 
```json
{
  "extends": "eslint-config-artifakt"
}
```
