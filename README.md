# ESLint Rules NodeJS

ESLint config file.

## Install

1. Install the config : 
```bash
$ npm install -D git+https://github.com/artifakt-io/eslint-config-artifakt.git#x.x.x
```

2. Create a `.eslintrc.json` file and copy/paste this `json` code block : 
```json
{
  "extends": "eslint-config-artifakt"
}
```
